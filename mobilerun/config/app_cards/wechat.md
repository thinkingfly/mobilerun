# 微信 WeChat Guide

## 基本信息
- 包名: `com.tencent.mm`
- 打开方式: 使用 `wechat_open` 或 `wechat_add_friend(phone_number)` 工具

## 安全限制（重要）
- **微信屏蔽了 Accessibility/UIAutomator 服务**，无法通过元素索引点击
- **必须使用 vision_only 模式**（截图 + 坐标点击）
- **ADB tap 对微信列表项无效** — 通讯录列表、弹窗菜单等都无法通过 ADB 点击
- **只能通过 vision 模式坐标点击** — Agent 需要根据截图自行判断坐标
- 输入文字通过 portal 键盘，ADB `input text` 无效

## 推荐操作流程（添加好友）
1. `wechat_add_friend("13800138000")` 打开微信
2. 切换到 vision_only 模式
3. 点击底部「通讯录」tab（底部第2个图标）
4. 点击「新的朋友」（列表第一项，橙色图标）
5. 点击顶部搜索框
6. 使用 `type_text` 输入手机号
7. 点击搜索结果
8. 点击「添加到通讯录」
9. 如有弹窗，按返回键或点击关闭

## 坐标参考（1260x2720 分辨率）
- 底部 tab 栏 y≈2640，4 个 tab 均分宽度
- 通讯录 tab: x≈472
- 搜索框: y≈160
- 列表项: 需要截图后自行判断

## 常见页面处理
- **已登录设备页面**: 安全验证页面，点击左上角 X 或按返回键关闭
- **备注和标签页面**: 添加好友后可能出现，点右上角"完成"
- **已添加好友页面**: 如果搜索手机号后，直接进入聊天窗口（底部有输入框可以发消息），说明该用户已经是好友。**此时立即调用 `complete` 标记任务成功**，不要再继续操作
- **搜索无结果**: 如果搜索手机号后没有匹配的用户，调用 `complete` 说明未找到该用户
- **申请添加朋友页面**: 如果点击搜索结果后进入"申请添加朋友"页面，显示用户信息和"发送"按钮。**点击"发送"后立即调用 `complete` 标记任务成功**，不要继续点击或等待后续页面

## 关键提醒
- 所有列表项必须通过 vision 模式坐标点击，不能用 ADB tap
- 输入文字使用 `type_text` 或 `type` action（不带 index），通过 portal 键盘
- **添加好友任务的结束条件**:
  1. 点击「添加到通讯录」后出现确认提示 → 任务完成，调用 `complete`
  2. 点击"发送"按钮（在"申请添加朋友"页面）→ **立即** `complete`，不要继续操作
  3. 搜索后直接进入聊天窗口（能发消息）→ 已是好友，任务完成，调用 `complete`
  4. 搜索后没有匹配结果 → 调用 `complete` 说明未找到
